Sidebar extension boilerplate for Opera

Quickly turn any responsively designed or mobile website into a sidebar extension for Opera. It takes only four simple steps:

1. Customize the extension name, description, and the URL in _locales/en/messages.js
2. Customize all the icons/*.png (square icons sized to match their filenames)
3. Drag manifest.js into the Extension Manager in Opera to test-install
4. Zip it all up and upload to addons.opera.com for users to download
